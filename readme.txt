 *  
 * Mechanisms:
 * Multiple Stages: 1-3 level, ballSpeed goes up when stage level becomes higher.
 * Score : 100 mark for each brick hit. No mark deduction in this game.
 * Game over after running out of 3 balls, user can choose play again or quit.
